# 客服关单评价自动核查系统

## 项目背景

作为一名客服，每天下班前必不可少的事情就是查询当天关闭的事件是否已有评价。

传统查询方式需要逐个点击事件号，再进入客户京豆记录，查看是否存在“满意度调研发放京豆”的说明，效率非常低。

为了减少重复人工操作，本工具提供一种更快捷的自动化查询方式，用于批量核查关单评价状态。

## 系统功能

1. **自动抓取关单数据**
   - 每日定时获取当日已关闭的事件列表。

2. **批量查询京豆记录**
   - 自动遍历事件，检索“满意度调研发放京豆”标记。

3. **智能筛选统计**
   - 区分已评价和未评价事件，并生成统计报表。

## 使用方法

1. 下载谷歌浏览器插件：
   - 链接地址：https://3.cn/11oKV-W69
   - 访问密码：`pfeuah`

2. 将 `JingDouQueryTool.zip` 解压到电脑桌面。

3. 打开 Google Chrome，依次进入：
   - 右上角三个点
   - 设置
   - 扩展程序
   - 打开右上角开发者模式
   - 点击【加载未打包的扩展程序】
   - 选择刚刚解压的文件夹

4. 打开 http://newadmin.jpos.jd.com/tool/beanList 网站。

5. 在页面中依次操作：
   - 点击【获取数据】按钮
   - 选择查询的【人员】
   - 选择请求来源【jpos/kfuad】
   - 点击【开始按钮】

> `jpos` 和 `kfuad` 两个请求来源都可以使用，`kfuad` 速度更快。
